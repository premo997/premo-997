# Personal Portfolio

A modern, responsive, and animated personal portfolio built using **HTML, CSS, JavaScript, and Three.js**.  
This project showcases my skills, projects, and creative frontend development style with a premium UI and interactive background effects.

---

## Features

- Modern and responsive design  
- Animated hero section with typing effect  
- Three.js creative animated background  
- Smooth scroll reveal animations  
- Skills, projects, gallery, and contact sections  
- Glass and gradient UI style  

---

##  Tech Stack

- HTML
- CSS
- JavaScript (ES6)  
- Three.js
